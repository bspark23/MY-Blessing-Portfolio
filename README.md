## ğŸ“ Customization

### Adding Your Own Content

1. **Personal Information**: Update the `PERSONAL_INFO` object in `portfolio-script.js`
2. **Projects**: Modify the `PROJECTS` array in `portfolio-script.js`
3. **Skills**: Update the `SKILLS` array in `portfolio-script.js`
4. **CV**: Replace `pdfs/cv.pdf` with your own CV
5. **Certificates**: Add your certificate PDFs to the `pdfs/` folder
6. **Images**: Replace profile images in the root directory

### PDF Files

- `cv.pdf` - Your main CV/Resume
- Certificate PDFs should be named exactly as specified in the code
- All PDFs are automatically served with proper headers
